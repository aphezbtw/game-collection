# GameCollection

[![Python](https://img.shields.io/badge/Python-3.11-blue)](https://python.org)
[![Flask](https://img.shields.io/badge/Flask-3.0.3-green)](https://flask.palletsprojects.com/)
[![License](https://img.shields.io/badge/License-MIT-yellow)](LICENSE)

**Live Demo**: [https://aphezbtw.pythonanywhere.com](https://aphezbtw.pythonanywhere.com)

Веб-приложение для коллекционирования видеоигр с возможностью добавления описаний, системных требований и инструкций по прохождению.

## Особенности

**Регистрация и авторизация** пользователей  
**Добавление и управление** играми  
**Поиск** по коллекции  
**Детальные страницы** с инструкциями  
**Адаптивный дизайн** для мобильных устройств  
**Развертывание** на PythonAnywhere  

## Быстрый старт

### Установка
```bash
# Клонируйте репозиторий
git clone https://github.com/aphezbtw/game-collection.git

# Установите зависимости
pip install -r requirements.txt

# Запустите приложение
python app.py
