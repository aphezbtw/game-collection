# GameCollection

Веб-приложение для коллекционирования видеоигр на Flask.

**Live Demo**: https://aphezbtw.pythonanywhere.com

## Функции:
- Регистрация и авторизация пользователей
- Добавление игр в коллекцию
- Поиск по играм
- Детальные страницы с инструкциями
- Адаптивный дизайн

## Технологии:
- Python 3.11
- Flask 3.0.3
- SQLite
- HTML/CSS
